# [OpenPanel](https://openpanel.dev/) Rust SDK

## Features

- Track events
- Identify users
- Increment and decrement properties

## Usage

- Set your OpenPanel API URL in the environment variable `OPENPANEL_TRACK_URL`
- Set your OpenPanel Client ID in the environment variable `OPENPANEL_CLIENT_ID`
- Set your OpenPanel Client secret in the environment variable `OPENPANEL_CLIENT_SECRET`

see tests for examples

## Testing

run `cargo test`
